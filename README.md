# Legal Docs Gen

RAG-powered generator for Terms of Service and Privacy Policy documents.

## Quick Start

1. Install dependencies:
```bash
uv pip install -r requirements.txt
```

2. Configure API key:
```bash
copy env.example .env
```
Edit `.env` and add your `OPENAI_API_KEY`.

3. Run:
```bash
python test_run.py
```
